# Mitwirken bei Askl-AI-on

Vielen Dank für Ihr Interesse, dieses Projekt zu unterstützen! Unser Ziel ist es, Ärzten einfache, sichere und kostenlose KI-Werkzeuge an die Hand zu geben.

## Wie Sie helfen können

Wir freuen uns über Beiträge jeder Art:
- **Fehler melden (Issues):** Wenn etwas nicht funktioniert, erstellen Sie ein Issue mit genauer Beschreibung und Fehlermeldung.
- **Verbesserungsvorschläge:** Haben Sie eine Idee für einen neuen Workflow oder ein besseres Prompt? Teilen Sie es!
- **Code-Beiträge (Pull Requests):**
    1. Forken Sie das Repository.
    2. Erstellen Sie einen neuen Branch (`git checkout -b feature/MeinNeuesFeature`).
    3. Fügen Sie Ihre Änderungen hinzu.
    4. Senden Sie einen Pull Request.

## Verhaltenskodex

Wir legen Wert auf einen respektvollen und konstruktiven Umgangston. Bitte bleiben Sie freundlich und hilfsbereit, da viele Nutzer (Ärzte) nicht unbedingt IT-Experten sind.

## Lizenz

Mit Ihrem Beitrag erklären Sie sich einverstanden, dass Ihre Änderungen unter der MIT-Lizenz dieses Projekts veröffentlicht werden.
